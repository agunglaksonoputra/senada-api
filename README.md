# 🌐 Backend API - SENADA

Ini adalah backend API untuk proyek SENADA yang dibangun menggunakan Node.js dan Express.js. API ini menyediakan endpoint untuk manajemen data.

---

## 🚀 Teknologi yang Digunakan

- [Node.js](https://nodejs.org/)
- [Express.js](https://expressjs.com/)
- [MySQL](https://www.mysql.com/)
- [dotenv](https://www.npmjs.com/package/dotenv)
- [nodemon](https://www.npmjs.com/package/nodemon)

---

```bash
git clone https://github.com/agunglaksonoputra/senada-api.git
